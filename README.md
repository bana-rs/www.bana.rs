## Build and Run

```sh
sudo dnf install perl-FindBin
curl -o zola.tar.gz -L "https://github.com/getzola/zola/releases/download/v0.18.0/zola-v0.18.0-x86_64-unknown-linux-gnu.tar.gz"
tar -xvf zola.tar.gz -C /usr/bin
zola serve
```
